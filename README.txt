This script sort out your tv shows.

Runs on at least Python 3.4.3

The script takes in two parameter: First: The name of the directory that you would like to be sorted.
Second: The name of the directory that you would like the sorted files to be placed in

Example:


>>> ls
    CleanUp.py	  downloads

>>> python CleanUp.py downloads Clean
    Done

>>> ls
    CleanUp.py    downloads   Clean
    

Packages used:
(os, re, shutil and sys)*

 * which are most likely included with python 3
